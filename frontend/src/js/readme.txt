1.Descripción de plugins:

- slick: Plugin de slide, se usa para crear todos los slides; slide logos de financieras.
- jquery.ddslick: Sirve para customizar los selects.
- jquery.validate: Sirve para validar todos los formularios.
- jquery.maskedinput: Placerholder interactivo para los campos de textos de formularios.
- kalypto: Sirve para customizar los checkbox.

------------------------------------------------------------------------------------------------
2.Descripción de módulos:
- b2b-nav: Código para los botones de navegación internos (B2B).
- buscador-entidades: Código para la caja de búsqueda de entidades financieras.
